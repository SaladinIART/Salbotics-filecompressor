# Salbotics File Compressor v0.2.0

This release expands Salbotics File Compressor from a basic PDF/JPG/PNG
compressor into a more capable document utility with optional external engines,
safer output decisions, better GUI feedback, and verified Windows packaging.

## Highlights

- Added Safe, Smart, and Aggressive quality modes.
- Added Force optimize for files already under the target size.
- Added optional qpdf and MuPDF mutool cleanup before Ghostscript fallback.
- Added optional ImageMagick routing for WebP, BMP, TIF, and TIFF images.
- Added safeguards so same-format compression keeps the original when every
  candidate would be larger.
- Added GUI result notes, engine readiness summary, and an Open output folder
  button.
- Added generated image smoke tests and release ZIP verification.
- Built and smoke-tested fresh Windows GUI and CLI executables.

## Engine Notes

- Ghostscript is required for real PDF compression.
- qpdf and MuPDF mutool are optional PDF cleanup helpers.
- ImageMagick is optional and only required for WebP/BMP/TIF/TIFF inputs.
- Pillow is bundled through the Python package dependency and handles JPG/JPEG/PNG.
