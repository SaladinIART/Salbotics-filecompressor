# Credits

Salbotics File Compressor uses Ghostscript for local PDF compression workflows,
Pillow for built-in image processing, and optional ImageMagick for extended
image formats.

## Ghostscript and Artifex Software

Ghostscript is developed and licensed by Artifex Software, Inc. This project
does not bundle Ghostscript; users install Ghostscript separately or point
`SALBOTICS_FILECOMPRESSOR_GS` at their local executable.
