# Security Policy

## Supported Versions

Salbotics File Compressor is pre-1.0. Security fixes target the latest source
version on the main branch.

## Reporting a Vulnerability

Please report sensitive security issues privately to the repository owner
instead of opening a public issue. Include:

- affected version or commit
- steps to reproduce
- expected impact
- sample file only if it is safe to share

## File Safety Notes

This app processes local documents. Avoid testing with confidential documents
unless you understand where output files are written. Ghostscript is an external
dependency and should be installed from official sources.
